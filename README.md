# Memo By Miraal Landing Page

A responsive fashion storefront with a warm, natural editorial direction and the supplied Memo by Miraal collection photography.

## Run locally

Open `index.html` directly, or serve the folder:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html` - semantic page structure and content
- `styles.css` - responsive design, layouts, and transitions
- `script.js` - mobile drawer, product tabs, wishlist, and newsletter interaction

All fashion imagery is stored locally in `assets/photos`.
